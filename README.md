## Descripción del Programa

Este programa fue desarrollado en C++ y permite calcular el total a pagar por la compra de un producto.  
Solicita el nombre completo del cliente utilizando `getline()`, así como el precio del producto y la cantidad comprada mediante `cin`.  
despues, calcula el subtotal, el IVA correspondiente al 12% y el total a pagar.  
El programa muestra los resultados con dos decimales utilizando `fixed` y `setprecision(2)`.  
No se utilizan estructuras de control, cumpliendo con lo pedido.
